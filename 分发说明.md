# 聚活 · 分发说明

本项目 `dist/` 目录有两个程序，选一个使用即可：

## 两个程序选哪个？

| 文件 | 大小 | 适用场景 |
|------|------|---------|
| `guyong-juhuo-1.0.0-setup.exe` | 46.5MB | 第一次用，想要桌面/开始菜单快捷方式 |
| `guyong-juhuo.exe` | 39.9MB | 不想安装，双击直接跑，关闭即退出 |

**两个程序功能完全一样，只是分发方式不同。**

---

## 安装包（推荐）

双击 `guyong-juhuo-1.0.0-setup.exe` → 下一步下一步 → 完成。

自动完成：
- 安装到 `C:\Program Files\聚活`
- 桌面创建快捷方式
- 开始菜单创建快捷方式
- 支持 Windows 控制面板卸载

## 免安装版

双击 `guyong-juhuo.exe` 直接运行，无需安装任何依赖。

```
guyong-juhuo.exe          启动网页控制台
guyong-juhuo.exe --help   显示帮助
guyong-juhuo.exe --tui    TUI 终端界面
guyong-juhuo.exe --check  安装前环境检测
```
